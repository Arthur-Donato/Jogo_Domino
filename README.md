# Projeto de Paradigmas de Programação

Este é o projeto final da disciplina Paradigmas de Programação, onde temos o jogo dominó realizado totalmente na linguagem LUA.

## 📥 Passo a Passo para Execução do Projeto

### 1. Instalação do Vscode
- Instale o Vscode pelo [link](https://code.visualstudio.com/) de acordo com seu sistema operacional.

### 2. Instalação do LOVE2D
- Instale o LOVE2D pelo [link](https://love2d.org/) também de acordo com seu sistema operacional.

### 3. Exportação do projeto
- Exporte esse projeto para alguma pasta dentro do seu computador da sua preferência;
- Caso queira testar o jogo com Banco de dados, na página inicial do projeto no GitHub mude a branch para "JogoComBancoDeDados".

## ▶️ Como Executar 

1. Dentro do seu Vscode procure a opção "Open Project" ou "Novo Projeto" e selecione a pasta Lua_Domino dentro da pasta na qual o projeto foi exportado;
2. Abra o terminal usando o comando "Ctrl + J" ou no menu do Vscode;
2. Dentro do terminal digite o seguinte comando "love ." e o jogo irá rodar.
